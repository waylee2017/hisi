Perform the following steps:

All test cases must be compiled in the upper-level directory (develop\sample).

Step 1 Open the sample folder, and run the make command.
Step 2 Run the following command:
./sample_psi  Frequency and symbol rate
or
./sample_psi  frequency
For example:
./sample_psi  403 6875
or
./sample_psi  403  (The symbol rate is 6875 by default.)
